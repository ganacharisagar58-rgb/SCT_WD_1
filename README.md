# Nexaura – Futuristic SaaS Landing Page

Nexaura is a futuristic cyberpunk-inspired SaaS landing page built using pure HTML, CSS, and JavaScript. It features a modern glassmorphism interface, animated particle backgrounds, neon glowing effects, responsive layouts, smooth scroll reveals, animated counters, testimonial sliders, interactive buttons, and immersive UI interactions designed to create a premium next-generation digital experience. The project combines futuristic aesthetics with responsive performance, making it ideal for startups, AI platforms, SaaS businesses, and modern tech product showcases.
